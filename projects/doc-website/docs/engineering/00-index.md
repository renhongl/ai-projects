## 学习建议
- 每读完一篇文档，都要在真实项目里落一次。
- 不要只记工具名字，要回答“这个工具解决什么问题”。
- 每个主题至少完成一次最小实践。
- 每周末复盘：我会用了什么，我理解了什么，我还能独立搭吗。
